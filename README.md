This project is a **React-based registration form** built using **Vite**.

The app demonstrates key React concepts such as:

- Conditional rendering for error messages  
- Form validation  
- Show/Hide password  
- Input formatting  
- Navigation using React Router  
- State transfer between routes  

---

##  Features

###  Form Validation
- First Name (required)  
- Last Name (required)  
- Username (required)  
- Email (format + required)  
- Password (required + show/hide toggle)  
- Phone Number with Country Code  
- Country & City (required)  
- PAN (valid format: ABCDE1234F)  
- Aadhaar (exactly 12 digits)  

###  Submit Button Disabled Until Valid  
The submit button remains disabled until **all fields are filled correctly**.
